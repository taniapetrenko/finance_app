# Персональний менеджер фінансів (ПМФ)
=============

#### Technologies Used
+ Ruby on Rails - 7.0.4
+ Ruby - 3.1.0
+ PostgresQL
+ Bootstrap

#### How to instal project

* Use ruby version `3.1.0`
* Run command `bundle install`
* Create database `rails db:create && rails db:migrate && rails db:seed`
